HTML 기초

HyperText Markup Language — 웹 페이지의 뼈대와 내용을 정의합니다

🏗️

기본 문서 구조

모든 HTML 문서는 정해진 뼈대를 가지고 있습니다. 브라우저가 이 구조를 읽어 화면에 그립니다.

HTML 
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
  </head>
  <body>
    <!-- 여기에 내용을 작성합니다 -->
  </body>
</html>
💡 <!DOCTYPE html>은 HTML5 문서임을 브라우저에 알려주는 선언문입니다
🏷️

CSS

CSS 기초

Cascading Style Sheets — HTML 요소의 디자인과 레이아웃을 담당합니다

🎯

선택자 (Selector)

스타일을 적용할 HTML 요소를 지정하는 방법입니다. 크게 3가지 방식이 있습니다.

CSS 
/* 태그 선택자 — 모든 p 태그 */
p { color: black; }

/* 클래스 선택자 — .highlight 클래스 */
.highlight { background: yellow; }

/* ID 선택자 — #title ID */
#title { font-size: 2rem; }

/* 후손 선택자 */
.card p { line-height: 1.6; }

/* 상태 선택자 (pseudo-class) */
button:hover { background: blue; }
💡 클래스(.class)는 여러 요소에, ID(#id)는 페이지 내 단 하나의 요소에만 사용하세요
📦

박스 모델 (Box Model)

모든 HTML 요소는 사각형 박스로 이루어져 있습니다. 4개의 영역으로 구성됩니다.

margin
border
padding
content
CSS 
.box {
  width: 200px;
  padding: 20px;         /* 내부 여백 */
  border: 2px solid #333; /* 테두리 */
  margin: 16px;          /* 외부 여백 */
  box-sizing: border-box; /* 권장 설정 */
}
💡 box-sizing: border-box를 쓰면 padding/border가 width에 포함되어 계산이 편해집니다
⚡

Flexbox 레이아웃

요소를 가로/세로로 자유롭게 배치할 수 있는 강력한 레이아웃 방식입니다.

1
2
3
CSS 
.container {
  display: flex;
  justify-content: center; /* 가로(주축) 정렬 */
  align-items: center;     /* 세로(교차축) 정렬 */
  gap: 16px;               /* 아이템 간격 */
  flex-wrap: wrap;         /* 줄 바꿈 허용 */
}
💡 위 버튼을 클릭하면 justify-content 값이 바뀌는 걸 직접 확인할 수 있습니다!
✨

호버 효과 (Hover)

마우스를 올렸을 때 시각적 변화를 주어 인터랙티브한 UI를 만듭니다.

마우스를 올려보세요 ✋
CSS 
.card {
  background: white;
  border-radius: 12px;
  transition: all 0.3s ease; /* 부드러운 전환 */
}

.card:hover {
  transform: translateY(-6px); /* 위로 이동 */
  box-shadow: 0 8px 24px rgba(0,0,0,0.15);
}
💡 transition을 꼭 추가해야 hover 시 부드럽게 애니메이션됩니다
🎨

CSS 변수 (Custom Properties)

자주 쓰는 값을 변수로 저장해 재사용할 수 있습니다. 다크모드 구현에도 핵심입니다.

CSS 
/* 변수 선언 (루트에 정의) */
:root {
  --primary: #3b82f6;
  --bg: #ffffff;
  --text: #111111;
}

/* 다크모드 변수 재정의 */
body.dark {
  --bg: #1a1a2e;
  --text: #e0e0e0;
}

/* 변수 사용 */
body {
  background: var(--bg);
  color: var(--text);
}
💡 :root에 변수를 선언하면 어디서든 var(--변수명)으로 참조할 수 있습니다
📱

반응형 디자인

화면 크기에 따라 다르게 보이도록 스타일을 분기하는 방법입니다.

CSS 
/* 기본: 1열 레이아웃 (모바일) */
.grid {
  display: grid;
  grid-template-columns: 1fr;
}

/* 768px 이상: 2열 레이아웃 (태블릿) */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: 1fr 1fr;
  }
}

/* 1024px 이상: 3열 레이아웃 (PC) */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
💡 모바일 우선(Mobile First) 방식으로 작성하는 것이 현대 웹 개발의 표준입니다

JavaScript 기초

웹 페이지에 동적인 기능과 상호작용을 추가합니다

📝

변수와 자료형

데이터를 저장하고 관리하는 기본 방법입니다. 변수 선언에는 3가지 키워드가 있습니다.

JS 
// 변수 선언
let name = "정선민";     // 변경 가능
const age = 20;         // 변경 불가 (상수)
var old = "구식 방식";  // 사용 비권장

// 자료형
let str = "문자열";       // String
let num = 42;            // Number
let bool = true;         // Boolean
let arr = [1, 2, 3];    // Array
let obj = { key: "값" }; // Object
💡 기본적으로 const를 쓰고, 값이 바뀌어야 할 때만 let을 사용하세요
⚙️

함수 (Function)

반복되는 코드를 묶어 재사용 가능하게 만드는 기본 단위입니다.

JS 
// 일반 함수 선언
function greet(name) {
  return `안녕하세요, ${name}님!`;
}

// 화살표 함수 (Arrow Function)
const greet = (name) => {
  return `안녕하세요, ${name}님!`;
};

// 단순한 경우 더 짧게도 가능
const add = (a, b) => a + b;

// 호출
console.log(greet("선민")); // 안녕하세요, 선민님!
console.log(add(3, 5));    // 8
💡 화살표 함수는 코드가 더 간결하고, 최신 JS에서 많이 사용하는 방식입니다
🖱️

DOM 제어

JavaScript로 HTML 요소를 찾고, 변경하고, 이벤트를 연결하는 방법입니다.

JS 
// 요소 선택
const btn = document.getElementById("btn");
const box = document.querySelector(".box");
const all = document.querySelectorAll(".card");

// 내용 변경
box.textContent = "새로운 텍스트";
box.innerHTML = "<strong>굵은 텍스트</strong>";

// 스타일 변경
box.style.color = "red";
box.style.fontSize = "20px";

// 클래스 제어
box.classList.add("active");
box.classList.remove("active");
box.classList.toggle("dark"); // 있으면 제거, 없으면 추가

// 이벤트 연결
btn.addEventListener("click", () => {
  console.log("버튼이 클릭되었습니다!");
});
💡 이 페이지의 다크모드 기능도 classList.toggle("dark")로 구현했습니다!
🔄

조건문과 반복문

조건에 따라 다르게 실행하거나, 코드를 반복 실행할 수 있습니다.

JS 
// 조건문 (if-else)
const score = 85;

if (score >= 90) {
  console.log("A 등급");
} else if (score >= 80) {
  console.log("B 등급"); // 출력됨
} else {
  console.log("C 이하");
}

// 반복문 (for)
const fruits = ["사과", "바나나", "딸기"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// 더 간결한 방식 (forEach)
fruits.forEach(fruit => {
  console.log(fruit);
});
💡 배열을 순회할 때는 전통적인 for문보다 forEach나 for...of를 선호합니다

/* 기본 (라이트 모드) */

  background: #ffffff;
  color: #111111;
}

/* 다크모드 클래스 적용 시 */
body.dark {
  background: #1a1a2e;
  color: #e0e0e0;
}
3

JS — 버튼 클릭 시 클래스 토글

const btn = document.getElementById("darkToggle");

btn.addEventListener("click", () => {
  // dark 클래스를 추가/제거 반복
  document.body.classList.toggle("dark");
});
