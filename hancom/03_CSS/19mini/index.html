구글 로고의 의미

구글 로고는 파란색, 빨간색, 노란색, 초록색으로 이루어져 있습니다.
여러 가지 색상을 사용하여 창의성과 다양성을 표현합니다.
심플한 디자인으로 누구나 쉽게 알아볼 수 있습니다.

HTML이란?

  • <h1> : 큰 제목
  • <p> : 문단 작성
  • <img> : 이미지 삽입
  • <a> : 링크 연결
  • <button> : 버튼 만들기
  • <ul>, <li> : 목록 만들기
  • <hr> : 구분선
  • <div> : 영역 구분

    • CSS 이란?

    • color : 글자 색
    • background-color : 배경 색
    • font-size : 글자 크기
    • text-align : 글자 정렬
    • margin : 바깥 여백
    • padding : 안쪽 여백
    • border : 테두리
    • border-radius : 둥근 모서리

    HTML 태그 퀴즈

    Q1. 제목을 만드는 태그는?

    ① <h1> ② <p>

    Q2. 문단을 만드는 태그는?

    ① <p> ② <img>

    Q3. 이미지를 넣는 태그는?

    ① <img> ② <a>

    Q4. 링크를 만드는 태그는?

    ① <a> ② <button>

    Q5. 버튼을 만드는 태그는?

    ① <button> ② <hr>

    정답

    1. <h1> / 2. <p> / 3. <img> / 4. <a> / 5. <button>
